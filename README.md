# Struktur-Data-A

1. Penjelasan Konsep Array
Array adalah struktur data yang digunakan untuk menyimpan sekumpulan data dengan tipe yang sama dan disimpan dalam memori yang berurutan (contiguous memory allocation).
Karakteristik array:
- Menyimpan banyak data dalam satu variabel
- Diakses menggunakan indeks
- Disimpan berurutan di memori

3. Analisis kompleksitas

| No | Operasi | Kompleksitas | Penjelasan |
|----|--------|-------------|-------|
| 1 | Input nilai | O(n) | Program membaca setiap nilai satu per satu |
| 2 | Nilai tertinggi | O(n) | Harus membandingkan seluruh elemen |
| 3 | Nilai terendah | O(n) | Harus memeriksa semua elemen |
| 4 | Rata-rata | O(n) | Penjumlahan semua elemen |
| 5 | Hitung kelulusan | O(n) | Mengecek setiap nilai apakah ≥ 60 |

Kesimpulan:
Semua operasi memiliki kompleksitas O(n) karena program harus memproses setiap elemen dalam array.

4. Refleksi pembelajaran
Melalui tugas ini saya memahami bahwa:

- Tugas ini juga memberikan pemahaman dasar tentang pengolahan sebuah data menggunakan struktur data yang sederhana.
